# Day 6 – Resume ↔ Job Description Matching

This phase adds job matching logic to the AI Resume Analyzer.

## Features
- Extracts skills from resume and job description
- Compares skills using set-based matching
- Calculates match percentage
- Identifies missing skills

## Why This Matters
This logic simulates how Applicant Tracking Systems (ATS)
evaluate resumes against job requirements.

## Technologies Used
- Python
- NLTK
